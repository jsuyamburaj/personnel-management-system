# Software Personnel Management System (SPMS)

A comprehensive full-stack personnel management system with modern UI, role-based access control, and complete HR, project, and task management features.

## Features

### Core Features
- ✅ Role-based Authentication (Admin, Manager, Developer, HR, Employee)
- ✅ Employee Management (CRUD with filters)
- ✅ Project Management with progress tracking
- ✅ Task Management with Kanban board
- ✅ Attendance Tracking (Check-in/out)
- ✅ Leave Management with approvals
- ✅ Payroll System
- ✅ Real-time Notifications
- ✅ Activity Logging
- ✅ Document Management
- ✅ Messaging System
- ✅ Analytics & Reports
- ✅ Performance Tracking

### Technology Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5, Chart.js, ApexCharts
- **Backend:** FastAPI (Python 3.9+)
- **Database:** MongoDB (with Motor async driver)
- **Authentication:** JWT + bcrypt password hashing
